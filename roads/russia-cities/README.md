# Крупные города России

Миниатюры дорог (trunk + primary + motorway) для городов с населением 300к+

## Содержимое

- `part-1.geojson` - Часть 1 (~19MB)
- `part-2.geojson` - Часть 2 (~19MB)  
- `part-3.geojson` - Часть 3 (~10-15MB)
- `all-cities-full.geojson` - Полная версия (для резервной копии)

## Использование

Файлы загружаются параллельно при zoom < 8 и отображаются на карте оранжевым цветом.

```typescript
// Автоматическая загрузка в MapView.tsx
loadRussiaMajorCities() 
```

## Генерация

```bash
python generate_russia_cities.py
```

Скрипт загружает данные из Overpass API с паузами 2 сек между запросами.
